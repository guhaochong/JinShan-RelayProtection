- 点表部分
  - 遥信点表、遥测点表 点号为 **0** 开始
  - 遥控点表 点号为 **1** 开始
- 配置部分
  - K型开关站 分为 **配变仓** 或 **站变仓** 配置，请根据现场实际情况进⾏配置报警灯部分

- 报警灯部分
  - 报警灯分2排或4排。2排为灯1~灯16；灯17~灯32既可
  - 报警灯2~14为触发事故总信
  - 报警灯4、5、6、7、8、12为保持信号，⼿动复归
